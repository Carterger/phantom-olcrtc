# 👻 Phantom

> Менеджер туннелей на основе WebRTC с поддержкой нескольких провайдеров потоковой передачи данных.

Phantom — десктопное приложение для управления серверной инфраструктурой OlcRTC. Позволяет разворачивать и управлять туннелями через популярные стриминговые платформы.

---

## Поддерживаемые провайдеры

- **Wildberries Stream** (`wbstream`) — стриминговая платформа Wildberries
- **SaluteJazz** (`salutejazz`) — стриминговая платформа СберДжаз
- **Яндекс Телемост** (`telemost`) — платформа видеоконференций Яндекс

---

## Возможности

- Установка серверной части одной кнопкой через SSH
- Создание и управление туннелями для нескольких пользователей
- Генерация QR-кодов и ссылок для подключения
- Мониторинг сервера в реальном времени (CPU, RAM, диск)
- Просмотр системных логов
- Автоматическое обновление ID комнаты без пересоздания туннеля
- Поддержка нескольких транспортных протоколов (datachannel, rudp)
- Многоязычный интерфейс (RU / EN)

## Быстрый старт

### Установка сервера одной командой

```bash
bash <(curl -fsSL https://raw.githubusercontent.com/Carterger/phantom-olcrtc/master/install.sh)
```

> Требования: Ubuntu 22.04, минимум 1GB RAM, 10GB диска

### При первом запуске приложения

1. Получите ID комнаты на платформе провайдера
2. Введите IP сервера и SSH данные
3. Нажмите **Установить сервер** (10-15 минут)
4. После установки создавайте клиентов и раздавайте ссылки

---

## Использование

### Создание клиента

1. Нажмите **+ Создать клиента**
2. Выберите провайдера и получите ID комнаты
3. Введите имя клиента
4. Нажмите **Создать** — получите QR-код и ссылку

### Подключение на устройстве

Пользователь устанавливает [Olcbox](https://github.com/openlibrecommunity/olcbox) и сканирует QR-код или вставляет ссылку формата:

```
olcrtc://wbstream?datachannel@{id}#{key}%{client_id}$OlcRTC
```

### Обновление ID комнаты

Нажмите **Обновить** рядом с клиентом, введите новый ID — туннель перезапустится автоматически.

## Технический стек

| Компонент | Технология |
|-----------|------------|

## Требования к серверу

| Параметр | Минимум |
|----------|---------|
| ОС | Ubuntu 22.04 |
| RAM | 1 GB |
| Диск | 10 GB |

Каждый туннель занимает ~30MB RAM. На 1GB сервере комфортно работают 15-20 туннелей одновременно.

## Разработка

```bash
git clone https://github.com/Carterger/phantom-olcrtc
cd phantom-olcrtc
npm install
npm run dev
```

Electron (два терминала):
```bash
npm run dev      # Терминал 1
npm run electron # Терминал 2
```
